# SoftwareHouse Backend - NODE / APOLLOSERVER / GRAPHQL / TYPESCRIPT / PRISMA / JEST

### :space_invader: Tech Stack

Clone the project

```bash
  git clone https://github.com/cristianoprogramador/BE_SH.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Linting

```bash
  npx eslint .
```

Test

```bash
  npm run test
```

Migrate for Prisma

```bash
  npm run migrate:dev
```
